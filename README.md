# PDB Big Data Project Data Visualization with React

Prerequisites:

-   Node Package Manager (npm)

---

## Initial Setup

### Run npm install:

-   Make sure you run this inside the project directory.

```
npm install
```

### Run the app:

-   Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
-   Afterwards, edit src/App.js and the page will reload once your edits are saved.

```
npm start
```
